Projects [edit]

Auto Mininet

Auto Mininet
Developer: Abdullah
Written in: Python, TypeScript
Platform: HuggingFace Spaces
Type: AI · Networking · LLM

Auto Mininet is an AI-powered network topology generation platform developed by Abdullah between 2025 and 2026.[4] The system accepts natural language prompts and converts them into runnable Mininet Python scripts using a fine-tuned large language model (LLM).

The platform includes a full-stack implementation featuring user authentication, chat history, an administrative panel, and a code output interface. The underlying model was trained on a custom dataset of Mininet topologies.[5]
